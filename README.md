# MatchResolution

## Overview
MatchResolution is a Python application designed to handle match resolution and data processing.

## Version
Current version: **1.0.1**

## Features
- Match resolution functionality
- CSV data processing
- Built-in version control

## Installation

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Setup
1. Clone or download the repository
2. Create a virtual environment:
   ```bash
   python -m venv .venv
   ```
3. Activate the virtual environment:
   - Windows: `.venv\Scripts\activate`
   - macOS/Linux: `source .venv/bin/activate`
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

### Running as Python Script
```bash
python MatchResolution.py
```

### Running as Executable (Windows)
```bash
MatchResolution.exe
```

## Building Executable

To build a standalone executable from the Python script:

```powershell
.\build_exe.ps1
```

The executable will be created in the `dist\` directory.
